# 📝 Online Examination System with Timer Using PHP & MySQL  

## 📌 Project Overview  
This project develops a web-based Online Examination System with Timer using PHP and MySQL. The system enables administrators to manage courses, exams, and students, while allowing students to take timed examinations online and view results instantly.

The project was created for learning purposes with guidance and support from YouTube tutorials.

## 🎯 Objectives  
• Develop a dynamic online examination platform  
• Implement timer-based exam functionality  
• Manage courses, exams, and students efficiently  
• Display automatic result calculation and ranking  

## 📊 System Modules  
• Admin Panel  
• Student Panel  
• Course Management  
• Exam & Questionnaire Management  
• Result & Ranking System  
• Feedback System  

## 🛠️ Key Functionalities  
• Create and manage courses  
• Add and manage exam questions  
• Timer-based exam attempt  
• Automatic submission after time completion  
• Instant result generation  
• Student feedback system  

## 🤖 Technologies Used  
• PHP  
• MySQL  
• HTML  
• CSS  
• Bootstrap  
• JavaScript  
• Ajax  
• jQuery  
• XAMPP  

## 📈 Features  
• Secure login for Admin and Students  
• Database-driven dynamic system  
• Session handling  
• Ranking by exam performance  
• User-friendly interface  

## 🚀 Deployment  
• Runs on XAMPP / WAMP / LAMP / MAMP  
• Database name: cee_db  
• Import provided SQL file in phpMyAdmin  

## 📌 Conclusion  
The project demonstrates the implementation of a full-stack web-based examination system using PHP and MySQL. It highlights database integration, timer-based logic, and result automation in an educational web application.

> ⚠️ Note: This system was developed for academic and learning purposes with the help of YouTube tutorials.
